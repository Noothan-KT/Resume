Technical Skills

HTML, or HyperText Markup Language, is the standard markup language used to create and design the structure of web pages. It forms the backbone of web development and is an essential skill for anyone involved in building websites.

CSS, or Cascading Style Sheets, is a stylesheet language used to control the presentation and layout of HTML documents. It enables web developers to define the visual style of web pages, including colors, fonts, spacing, and positioning of HTML elements.

C is a general-purpose, procedural programming language that has been widely used for developing a variety of applications, including system software, embedded systems, and application software.

JavaScript is a versatile and widely-used programming language primarily known for its ability to create dynamic and interactive content on websites
